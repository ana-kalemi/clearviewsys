What is CXR Cloud?

CXR Cloud is a revolutionary business app for Currency Exchange and Money Transfer Organizations (MTO). It is a web-based application and has been developed based on the popular CurrencyXchanger desktop edition. CXR Cloud can capture day to day activities of an exchange office, from the front office desk to the back office.

Qu'est-ce que CXR Cloud ?

CXR Cloud est une application commerciale révolutionnaire pour les organisations de change et de transfert d'argent (MTO). Il s'agit d'une application Web et a été développée sur la base de la populaire édition de bureau CurrencyXchanger. CXR Cloud peut capturer les activités quotidiennes d'un bureau de change, de la réception au back-office.

¿Qué es la nube CXR?

CXR Cloud es una aplicación comercial revolucionaria para organizaciones de cambio de divisas y transferencia de dinero (MTO). Es una aplicación basada en web y ha sido desarrollada en base a la popular edición de escritorio de CurrencyXchanger. CXR Cloud puede capturar las actividades diarias de una oficina de cambio, desde el mostrador de recepción hasta la oficina administrativa.

Was ist CXR-Cloud?

CXR Cloud ist eine revolutionäre Business-App für Währungsumtausch- und Geldtransferorganisationen (MTO). Es ist eine webbasierte Anwendung und wurde basierend auf der beliebten Desktop-Edition von CurrencyXchanger entwickelt. CXR Cloud kann die täglichen Aktivitäten einer Wechselstube erfassen, von der Rezeption bis zum Backoffice.
